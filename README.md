# App KUD

Početna Next.js web aplikacija za upravljanje kulturno-umetničkim društvima.

## Šta je trenutno urađeno

- Osnovna Next.js struktura
- Početna strana
- Osnovni izgled
- Spisak budućih modula

## Budući moduli

- Login / logout
- Registracija društva
- Master admin i odobravanje
- Moji podaci
- Dashboard
- Članovi
- Sekcije
- Prisustvo
- Finansije
- Izveštaji
- Podešavanja
